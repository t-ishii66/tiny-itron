# include/itron.h - ITRON API 型定義とファンクションコード

## 概要

Micro ITRON v4.0.0 仕様の型定義、エラーコード、属性定数、ファンクションコードを定義するヘッダファイル。
カーネルおよびユーザータスクの両方から参照される、ITRON API の中核的なヘッダである。

末尾で `types.h` を `#include` しており、構造体定義も間接的に取り込まれる。

## 基本データ型

### プリミティブ型

| 型名 | C 定義 | 説明 |
|------|--------|------|
| `B` | `char` | 符号付き 8 ビット |
| `UB` | `unsigned char` | 符号なし 8 ビット |
| `H` | `short` | 符号付き 16 ビット |
| `UH` | `unsigned short` | 符号なし 16 ビット |
| `W` | `long` | 符号付き 32 ビット |
| `UW` | `unsigned long` | 符号なし 32 ビット |

| 型名 | C 定義 | 説明 |
|------|--------|------|

## API 関数宣言 (条件コンパイルにより無効化)

ヘッダ内に `#if 0` で囲まれた API 関数プロトタイプが存在する。
実際のユーザー API はライブラリ層 (`lib/`) で `int 0x80` syscall 経由のラッパーとして提供される。

主な関数:
- タスク管理: `cre_tsk`, `del_tsk`, `act_tsk`, `sta_tsk`, `ext_tsk`, `exd_tsk`, `ter_tsk`, `chg_pri`, `get_pri`, `ref_tsk`, `ref_tst`
- 起床・待ち: `slp_tsk`, `tslp_tsk`, `wup_tsk`, `can_wup`, `rel_wai`, `sus_tsk`, `rsm_tsk`, `frsm_tsk`, `dly_tsk`
- CPU 制御: `loc_cpu`, `unl_cpu`
