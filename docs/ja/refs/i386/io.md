# io.md

対象ファイル: `i386/io.h`

## 概要

x86 I/O ポートアドレスの定数定義。カーネルが直接操作する各種ハードウェアデバイス (PIC、PIT、キーボード、FDC、DMA、VGA、NE2000) のポートアドレスを集約する。

ファイル先頭には割り込みベクタ番号の定義 (BIOS 標準) も含まれているが、カーネルの割り込みベクタリマッピング後はこれらは直接使用されない (`interrupt.h` の `VECT_IRQ*` 定数が使用される)。

## 定数・マクロ

### 割り込みベクタ番号 (BIOS 標準)

| 定数 | 値 | 説明 |
|------|------|------|
| `VINTR_TIMER` | `0x08` | タイマー (IRQ0) の BIOS デフォルトベクタ |

## 補足

### I/O ポートアクセス関数

I/O ポートへのアクセスは以下のインラインアセンブリ関数で行われる (klib.h 等で定義):

- `outb(port, val)` -- 指定ポートに 1 バイト出力
- `inb(port)` -- 指定ポートから 1 バイト入力

### ポートアドレスの慣例

- IBM PC/AT 互換機の標準 I/O ポートアドレスに準拠
- DMA コントローラのチャネル 2 は FDC 専用として使用
- VGA テキスト VRAM (0xB8000) は I/O ポートではなくメモリマップド I/O でアクセスされる

| デバイス | ドライバファイル | 主要ポート |
|----------|-----------------|------------|
