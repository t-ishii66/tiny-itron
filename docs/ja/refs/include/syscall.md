# include/syscall.h - Syscall 定数

## 概要

syscall 関連の定数を定義するヘッダファイル。
現在の実装では 1 つの定数のみ定義されており、実際の syscall ディスパッチには使用されていない。

## 定数

| 定数 | 値 | 説明 |
|------|--------|------|
| `SC_CRE_TSK` | `0x1234` | タスク生成 syscall コード |

## 備考

このヘッダの定数は現在のカーネルでは使用されていない。
実際の syscall ディスパッチは `itron.h` で定義されるファンクションコード (`TFN_CRE_TSK` = `-0x05` 等) に基づいて行われる。

syscall の流れ:
1. ユーザータスクが `lib/` のラッパー関数を呼ぶ
2. ラッパーがファンクションコードを EAX にセットし `int 0x80` を発行
3. `i386/syscall.c` の `c_intr_syscall` がファンクションコードに基づいてカーネル関数をディスパッチ
4. `kernel/syscall.c` の関数テーブルで実際のカーネル関数が呼ばれる
